# Refunds and Cancellations

## Cancellation Policy

| Timing | Refund |
|---|---|
| Cancelled before rental partner confirms | Full refund |
| Cancelled more than 48 hours before start | Full refund |
| Cancelled 24 to 48 hours before start | Partial refund |
| Cancelled less than 24 hours before start | No refund |

## Refund Processing

- Refunds are sent back to the original payment method.
- Processing speed depends on the payment provider and bank or wallet timelines.

## Disputes

If cancellation or delivery issues are contested, admin review is used to determine final resolution.

# System Overview

RentalBasic connects customers, rental partners, and admins in a single workflow:

1. Customers discover rental partners and place bookings.
2. Rental partners confirm, deliver, and complete fulfillment.
3. Customers confirm delivery.
4. Payouts are released based on platform rules.
5. Admins oversee quality, compliance, and disputes.
